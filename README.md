# Node repo starter
Kick-start Node.JS repo with minimal setup for Test-driven development based workflow:
- Linter
- Nodemon
- Mocha
- Babel compiler

workflow/Gulp tasks:

`lint -> compile -> test -> start nodemon -> watch for changes -> repeat`

## Usage:

1. git clone
2. npm i
3. gulp
